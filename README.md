# Boat-Sales-Analysis-using-PowerBI
This project presents an end-to-end Boat Sales Analytics Dashboard built using Power BI to analyze customer behavior, revenue performance, product trends, and returns. The dashboard helps stakeholders monitor key business metrics and make data-driven decisions efficiently.<br>

# Dashboard Overview -
1. Customers Analysis -
   - 12K+ total customers from 20 cities
   - Customer distribution by age group & gender
   - Monthly customer login trends
2. Orders & Revenue -
   - ₹105M+ total revenue
   - 89K+ units sold
   - Order status breakdown
   - Revenue by payment method
   - Top 5 revenue contributing states
3. Product Performance -
   - 379 total products across 10 categories
   - Top selling vs lowest selling products
   - Revenue contribution by product category
4. Returns Analysis -
   - 4.5K total returns
   - Return status tracking (Approved / Processed / Pending)
   - Returns by month, quarter, and category

# Project Objective - <br>
✔️ Track and visualize overall sales performance <br>
✔️ Understand customer behavior and demographics <br>
✔️ Identify top-performing regions & products <br>
✔️ Monitor returns and operational efficiency <br>
✔️ Support data-driven decision making <br>

# Tools & Technologies -
- Power BI
- DAX
- Data Cleaning & Modeling
